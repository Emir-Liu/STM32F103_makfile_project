# Introduce

This is a temple project build by makefile for STM32F103.

And most of the files is coming from the STM32F10x_StdPeriph_Lib_V3.5.0 and modified by myself.

the tools used are as followed:

makefile
gcc-arm-none-eabi
stm32flash(ing)
openocd(ing)
minicom(ing)
